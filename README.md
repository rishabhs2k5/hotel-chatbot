# 🏨 HotelBot – Flask Chatbot

A conversational hotel chatbot built using Flask that helps users explore hotel details like rooms, pricing, amenities, services, and more — all through a chat interface.

---

## 🚀 Features

- 💬 Chat-based interaction (no forms)
- 🌍 Multi-city hotel support
- 🏨 Detailed hotel information (rooms, prices, amenities, etc.)
- ⚡ Real-time responses using Flask backend
- 🎨 Modern UI with chat widget

---

## 🛠 Tech Stack

- Python (Flask)
- HTML
- CSS
- JavaScript

## ▶️ How to Run Locally

1. Install Flask:
pip install flask  

2. Run the app:
python chatbot.py  

3. Open in browser:
http://127.0.0.1:5000/
